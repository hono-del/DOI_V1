# Lean3 オーナーズマニュアル企画書

## プロジェクト概要

**対象製品**: Lean Mobility「Lean3」都市型小型EV  
**参考**: [Lean Mobility公式サイト](https://leanmobility.net/jp/)  
**企画日**: 2026年2月16日

---

## 製品特性と制約条件

### 製品の特徴
- 一人乗り（日本市場）/ 二人乗り（台湾市場）
- 通常の乗用車の約1/3のサイズ
- Active Lean System搭載
- フル視界キャビン、エアコン完備
- 機能は非常にシンプル

### 制約条件
- ✗ **紙をしまう場所がない** - 車内が極めて狭い
- ✗ **従来型マニュアルは不適合** - 分厚い冊子は保管不可
- ○ **初めて乗る人が多い** - 直感的な理解が必要
- ○ **新しい体験の創造** - 革新的なアプローチが求められる

### ブランドビジョン
> 「美しい、ワクワク、楽しい。人間の本能に訴える価値を、モビリティを通して多くの人に提供し続けたい」  
> — 谷中壯弘 代表取締役社長

## 提案2: デジタルオーナーズエクスペリエンス

### A. 専用Webアプリ「Lean3 Guide」

#### アクセス方法
1. **車内QRコード** - ダッシュボードに常設
2. **スマートカードQRコード** - 個人専用URL
3. **車両Bluetooth** - 近づくと自動通知（オプション）

#### 構成

##### 🏠 ホーム画面
```
┌─────────────────────────────┐
│  Welcome to Lean3           │
│  [車両3Dビジュアル]           │
│                             │
│  🎯 Quick Actions           │
│  ├─ First Ride Tutorial     │
│  ├─ Today's Tips            │
│  └─ Find Charging Station   │
│                             │
│  📚 Guide Menu              │
│  ├─ Getting Started         │
│  ├─ Features & Controls     │
│  ├─ Maintenance             │
│  ├─ Troubleshooting         │
│  └─ Community               │
└─────────────────────────────┘
```

##### 🎓 First Ride Tutorial（初回乗車ガイド）
- **動画ベース**: 各30秒以内の短編動画
- **インタラクティブ**: タップして次へ進む
- **内容**:
  1. 乗り込み方（ドアの開け方、シートポジション）
  2. 基本操作（パワーON、ドライブモード選択）
  3. Active Lean Systemの体感
  4. 駐車とロック
  5. 充電方法

##### 📖 Features & Controls（機能と操作）
- **3Dインタラクティブビュー**: 
  - 車両を360度回転
  - 各パーツをタップすると説明表示
  - AR機能で実車に重ねて表示（オプション）

- **カテゴリー**:
  - ドライビング
  - Active Lean System
  - 充電
  - コネクティビティ
  - 安全機能

##### 🔧 Maintenance（メンテナンス）
- **メンテナンススケジュール**: 走行距離連動
- **DIYガイド**: タイヤ空気圧チェック、ウォッシャー液補充など
- **サービス予約**: ワンタップで最寄りサービスセンター予約

##### ⚠️ Troubleshooting（トラブルシューティング）
- **症状から検索**: 「音がする」「動かない」など
- **エラーコード解説**: 警告灯の意味
- **緊急時対応**: 
  - 事故時の手順
  - ロードサービス呼び出し
  - 緊急連絡先

##### 👥 Community（コミュニティ）
- **オーナーズストーリー**: 他のオーナーの使い方
- **おすすめルート**: 都市別のドライブコース
- **Tips & Tricks**: 上級者向けテクニック

---

### B. AR（拡張現実）マニュアル機能

#### 「Lean3 AR Assistant」

スマートフォンを車両にかざすと、リアルタイムで情報を表示

**使用例**:
1. **初回乗車時**: 
   - スマホを車内にかざす
   - 各操作部に説明が浮かび上がる
   - 「ここを押してスタート」と矢印表示

2. **充電時**:
   - 充電ポートにかざす
   - 接続手順がステップバイステップで表示

3. **メンテナンス時**:
   - タイヤにかざすと空気圧の確認方法を表示

## リスクと対策

### リスク1: デジタルデバイド
**懸念**: 高齢者などスマホ不慣れな層  
**対策**: 
- 納車時の丁寧なサポート
- 電話サポート充実
- オプションでEssentials Card提供

### リスク2: 通信環境
**懸念**: 圏外でアクセスできない  
**対策**:
- PWAオフライン対応
- Smart Cardに最重要情報
- 車内ディスプレイにオフラインガイド

### リスク3: 情報更新の遅れ
**懸念**: コンテンツが古くなる  
**対策**:
- 専任コンテンツマネージャー配置
- オーナーフィードバック収集
- 四半期ごとの定期更新
