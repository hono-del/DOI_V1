テーマ: スマートキーでドアが開かない（実践版）
ID: THEME-001
緊急度: high
インテント: I1, I3, I4

▼ スマートキーでドアが開かない

  ? スマートキーのインジケーターは点灯する?
    → いいえ: キー電池切れ疑い
    → はい: 車両側問題疑い

■ キー電池切れ疑い
  
  ? どうしたい?
    → すぐ開けたい: 機械キー使用
    → 電池交換したい: 電池交換

  ■ 機械キー使用
    
    ? 機械キーはある?
      → はい: 機械キー取り出し
      → いいえ: 機械キー未所持
    
    ■ 機械キー取り出し
      [GUIDE] 機械キーの取り出し方
      
      ? 機械キーで開錠できた?
        → はい: ✓ 解決
        → いいえ: ドアロック故障
    
    ■ 機械キー未所持
      [GUIDE] 機械キーの場所・保管方法
      → ⚠ ロードサービス

  ■ 電池交換
    
    ? どうしたい?
      → 自分で交換: 電池交換準備
      → 販売店で交換: 販売店予約
    
    ■ 電池交換準備
      [GUIDE] 事前準備物（マイナスドライバー・小さいマイナスドライバー・CR2450電池）
      
      ? 準備物は揃ってる?
        → はい: 電池交換実施
        → いいえ: 電池購入サポート
    
    ■ 電池購入サポート
      [GUIDE] 電池購入方法（オンライン購入・近くの店舗検索）
      
      ? 電池を入手した?
        → はい: 電池交換実施
        → いいえ: 販売店予約
    
    ■ 電池交換実施
      [GUIDE] 電池交換手順（注意: 部品破損のおそれあり）
      
      ? 電池交換後、解決した?
        → はい: ✓ 解決
        → いいえ: 電池交換でも未解決
    
    ■ 販売店予約
      [GUIDE] 販売店予約・連絡先
      → ⚠ 販売店対応
    
    ■ 電池交換でも未解決
      
      ? 車両の電装品（ライト等）は動作する?
        → いいえ: バッテリー上がり
        → はい: 電波干渉診断

■ 車両側問題疑い
  
  ? 車両の電装品（ライト等）は動作する?
    → いいえ: バッテリー上がり
    → はい: 電波干渉診断

■ バッテリー上がり
  
  ? ジャンプスタートできる環境?
    → はい: ジャンプスタート実施
    → いいえ: ⚠ ロードサービス
  
  ■ ジャンプスタート実施
    [GUIDE] ジャンプスタート手順
    → ✓ 解決

■ 電波干渉診断
  
  ? 強い電波を発する施設（空港・放送局等）の近く?
    → はい: 電波干渉対応
    → いいえ: その他診断
  
  ■ 電波干渉対応
    [GUIDE] 電波干渉時の対応（キーを車体に近づける）
    
    ? 解決した?
      → はい: ✓ 解決
      → いいえ: その他診断

■ その他診断
  
  ? 予備のキーはある?
    → はい: 代替キー使用
    → いいえ: ✗ ドアロック故障
  
  ■ 代替キー使用
    
    ? 予備のキーで解決した?
      → はい: 元のキー故障
      → いいえ: ✗ ドアロック故障

■ 元のキー故障
  [GUIDE] キー修理・交換手続き
  → ⚠ 販売店対応

✗ ドアロック故障
  [GUIDE] ドアロック故障時の対応
  → ⚠ ロードサービス

⚠ ロードサービス
  [GUIDE] ロードサービス連絡先・依頼方法
  → (終了)

⚠ 販売店対応
  [GUIDE] 販売店連絡先・営業時間
  → (終了)

✓ 解決
  [GUIDE] 予防策・定期点検の案内
  → (終了)
