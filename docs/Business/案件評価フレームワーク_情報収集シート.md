## 📊 評価基準の詳細説明

### **1. ビジネス情報**

#### **契約金額の考え方**
- **初期費用：** システム開発、導入、カスタマイズ費用
- **年間費用：** 月額利用料、保守費用、API利用料等
- **3年間累計：** 初期費用 + 年間費用×3年

#### **契約形態の種類**
- **SaaS型：** 継続課金、アルゴリズムは当社資産
- **買い取り型：** 一括払い、システムは顧客資産
- **コンサル型：** ノウハウ提供、顧客が自社開発

---

### **2. 成功確率評価（各項目1-10点）**

| 項目 | 評価基準 |
|:--|:--|
| **関心度** | 10点：強い関心、具体的な相談あり<br>5点：一般的な関心<br>1点：関心薄い |
| **予算確保** | 10点：予算確保済み<br>5点：予算申請中<br>1点：予算未定 |
| **意思決定権限** | 10点：キーパーソンと直接対話<br>5点：担当者レベル<br>1点：窓口不明 |
| **競合状況** | 10点：競合なし<br>5点：競合あり、差別化可能<br>1点：競合優位 |
| **技術的実現性** | 10点：技術的に確実<br>5点：一部課題あり<br>1点：技術的に困難 |
| **スケジュール適合** | 10点：顧客スケジュールと完全一致<br>5点：調整可能<br>1点：スケジュール厳しい |

#### **成功確率の計算**
```
PoC成功確率 = (関心度 + 予算確保 + 意思決定権限 + 競合状況 + 技術的実現性 + スケジュール適合) ÷ 60 × 100%

本格展開確率 = PoC成功確率 × 0.8（PoC成功を前提）
```

---

### **3. 戦略的価値（各項目1-10点）**

| 項目 | 評価基準 |
|:--|:--|
| **事例価値** | 10点：業界標準になる可能性<br>5点：他案件への参考になる<br>1点：個別案件で完結 |
| **技術蓄積** | 10点：新技術の獲得<br>5点：既存技術の応用<br>1点：技術的学びなし |
| **市場拡大性** | 10点：大規模な横展開可能<br>5点：一定の横展開可能<br>1点：横展開困難 |
| **継続性** | 10点：長期継続収益確実<br>5点：継続の可能性あり<br>1点：単発案件 |
| **ブランド価値** | 10点：企業ブランド大幅向上<br>5点：一定の貢献<br>1点：ブランド価値低い |

#### **戦略的価値スコアの計算**
```
戦略的価値スコア = (事例価値 + 技術蓄積 + 市場拡大性 + 継続性 + ブランド価値) ÷ 50 × 10点
```

---

### **4. リスク評価**

#### **影響度の定義**
- **高：** プロジェクト失敗、大幅な損失
- **中：** プロジェクト遅延、一部損失
- **低：** 軽微な影響

#### **発生確率の定義**
- **高：** 50%以上
- **中：** 20-50%
- **低：** 20%未満

#### **リスクスコアの計算**
```
リスクスコア = Σ(影響度 × 発生確率)
- 影響度：高=3、中=2、低=1
- 発生確率：高=3、中=2、低=1
- リスクスコアが高いほど、リスクが大きい
```

## 🎯 総合評価の計算方法

### **総合スコアの計算式**

```
総合スコア = (収益スコア × 30%) + (成功確率スコア × 25%) + (戦略的価値スコア × 25%) + (スピードスコア × 20%) - (リスクスコア × 10%)
```

### **各スコアの計算方法**

#### **1. 収益スコア（10点満点）**
```
収益スコア = (初年度収益見込み ÷ 1億円) × 10点
※ 1億円以上は10点満点
```

#### **2. 成功確率スコア（10点満点）**
```
成功確率スコア = (PoC成功確率 × 0.5 + 本格展開確率 × 0.5) ÷ 10
```

#### **3. 戦略的価値スコア（10点満点）**
```
戦略的価値スコア = (事例価値 + 技術蓄積 + 市場拡大性 + 継続性 + ブランド価値) ÷ 50 × 10点
```

#### **4. スピードスコア（10点満点）**
```
スピードスコア = 10 - (契約までの月数 ÷ 12 × 10)
※ 1ヶ月で契約：10点、12ヶ月で契約：0点
```

#### **5. リスクスコア（減点方式）**
```
リスクスコア = Σ(影響度 × 発生確率) ÷ 最大リスクスコア × 5点
※ 最大リスクスコア = 主要リスク数 × 9（高×高=3×3）
```
