**具体例:**

```

【予測のみの場合】
システム: 「雨が降ります」
→ ワイパーの使い方がわからない（情報不足）

【RAG + 予測の統合】
システム: 「雨が降ります。オートワイパーをONにしますか？
         レバーはここです（AR表示）」
→ 困る前に、正確な情報を提供（理想の体験）
```

```

├─ スマホアプリ（AR対応）
├─ 音声アシスタント
└─ ウェアラブル

├─ ユーザーフィードバック

└─ 継続的改善
```
