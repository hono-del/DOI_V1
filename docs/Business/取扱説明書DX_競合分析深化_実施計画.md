**担当**: 事業企画部門、マーケティング部門

## 1. 目的とスコープ

### 1.1 目的
- 取扱説明書DX事業の**市場ポジション**と**差別化の根拠**を明確にする
- 競合参入・価格競争リスクを**定量・定性**で把握し、意思決定に反映する
- **競合対抗策**と**参入タイミング仮説**を策定し、事業計画・営業戦略に組み込む

### 1.2 スコープ
- **直接競合**: 取扱説明書デジタル化・車載マニュアル系サービス
- **間接競合**: 車載AI・音声アシスタント、コネクテッドカー／インフォテインメント
- **潜在競合**: 大手クラウド・AIプロバイダーの車載向けサービス、OEM内製

### 1.3 成果物一覧
| No | 成果物 | 形式 | 期限 |
|----|--------|------|------|
| 1 | 競合一覧・分類マトリクス | Excel/CSV + MD | Week1末 |
| 2 | 競合個別分析シート（各競合1枚） | MD/Excel | Week2末 |

## 2. フェーズ別実施計画

### Phase 1: 競合の特定と分類（Week 1）

#### 1.1 競合候補の洗い出し
- [ ] **一次リスト作成**
  - キーワード: 取扱説明書 デジタル化、車載 マニュアル、車載AI、インフォテインメント、コネクテッド マニュアル
  - 調査手段: Web検索、業界レポート、展示会出展社、特許・ニュース検索
- [ ] **社内ヒアリング**
  - 営業・技術・事業開発に「競合として認識しているサービス・企業」を聴取
- [ ] **OEM・Tier1視点の確認**
  - 既存関係者への簡易ヒアリング（可能な範囲）：採用済み／検討中の類似サービス

#### 1.2 分類と優先順位
- [ ] **3層分類**
  - **直接競合**: 取説デジタル化・車載マニュアルが主サービス
  - **間接競合**: 車載AI・コネクテッドの一機能として取説／マニュアルを提供
  - **潜在競合**: 現時点で取説特化ではないが、参入可能性が高いプレイヤー
- [ ] **分析優先度の設定**
  - 直接 → 間接 → 潜在の順で深掘り優先
  - 市場シェア・顧客数・直近動向で「重点競合」を5–10社に絞る

#### 1.3 成果物（Week 1）
- **競合一覧・分類マトリクス**（会社名、サービス名、分類、優先度、情報ソース）
- **情報ソース一覧**（URL、レポート名、日付）の記録

---

### Phase 2: 各競合の詳細分析（Week 2）

#### 2.1 分析項目（全重点競合に共通）
各競合について以下を整理する。

| 項目 | 内容 | 情報源例 |
|------|------|----------|
| サービス内容・機能 | 提供機能、UI形態（音声/画面）、対象車種・OEM | 公式サイト、プレスリリース、デモ動画 |
| 価格設定 | 料金体系、単価感、無料枠の有無 | 公式、問い合わせ、業界情報 |
| 顧客基盤 | 採用OEM・車種、契約数・車両数（推定可） | プレスリリース、IR、業界紙 |
| 技術的優位性 | 技術スタック、AI/LLM利用、オフライン対応等 | 技術ブログ、特許、カンファレンス |
| 市場シェア | 車両台数ベース・契約数ベースの推定 | 自社推定＋レポート |
| 最近の動向 | 新機能、パートナー、M&A、採用拡大（1–2年） | ニュース、SNS、IR |

#### 2.2 情報収集ルール
- [ ] 事実と推計を区別して記載（出典・日付を明記）
- [ ] 価格・数値は「〜程度」「推定」と表記し、更新日を記載
- [ ] 週次で「新着動向」を1行で追記する枠を用意

#### 2.3 成果物（Week 2）
- **競合個別分析シート**（重点競合ごとに1ファイルまたは1シート）
- **競合比較表**（横軸: 競合、縦軸: 上記項目）— 機能・価格・顧客基盤は表形式で一覧化

---

### Phase 3: 影響分析と競合シナリオ（Week 3 前半）

#### 3.1 競合参入シナリオの設定
- [ ] **シナリオの定義**
  - シナリオA: 既存の直接競合が価格引き下げ・機能強化
  - シナリオB: 大手クラウド/AIベンダーが車載取説ソリューションを提供
  - シナリオC: 主要OEMが自社内製で同等機能を開発
- [ ] **各シナリオの想定**
  - 発生確率（低/中/高）
  - 想定タイミング（1年以内／1–3年／3年以降）
  - 自社への影響（市場シェア、単価、受注機会）

#### 3.2 影響の定量化（可能な範囲）
- [ ] **市場シェアへの影響**
  - 各シナリオで想定される自社シェア変動（%または件数）
- [ ] **価格競争への影響**
  - 価格下落の幅の仮説（例: 〜%下落）
- [ ] **顧客獲得への影響**
  - 獲得難易度の変化、RFPでの競合出現頻度の仮説

#### 3.3 成果物（Week 3 前半）
- **影響分析レポート**（シナリオ別の記述＋簡易表）
- **リスクマトリクス**（発生確率×影響度）の更新用入力

---

### Phase 4: 競合対抗策と参入タイミング（Week 3 後半）

#### 4.1 競合対抗策の策定
- [ ] **技術的差別化**
  - 自社の強み（RAG最適化、安全性、データ還流等）の整理
  - 競合が真似しにくい要素の明確化
- [ ] **価格戦略**
  - プレミアム戦略 vs 浸透価格の選択肢と条件
  - バンドル・長期契約によるロックイン案
- [ ] **パートナーシップ**
  - Tier1・車載OS・クラウドとの連携で差別化するポイント
- [ ] **ブランド・信頼**
  - 「安全性」「取説の正確性」を前面に出す打ち手

#### 4.2 参入タイミングの分析
- [ ] **各競合の参入可能性**
  - 製品ロードマップ、投資動向、採用情報から推測
- [ ] **参入タイミングの予測**
  - 短期（1年以内）／中期（1–3年）／長期（3年以降）の整理
- [ ] **自社の先行優位を活かす施策**
  - 先行顧客の獲得、実績・ケーススタディの蓄積、規格・ガイドラインへの関与

#### 4.3 成果物（Week 3 後半）
- **競合対抗策・参入タイミング分析**（MD、施策一覧と優先度）
- **競合分析サマリー**（経営・意思決定用 2–3頁）
  - 主要競合の整理、リスク要約、推奨アクション3–5項目
