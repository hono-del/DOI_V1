## 💡 通常版とハンズフリー版の違い

### **推奨の優先順位**

```
同じシーンで両方の機能が装備されている場合：

1. ハンズフリーパワーバックドア（FN016）を優先推奨
   ↓
2. 通常のパワーバックドア（FN009）は推奨しない
   （ハンズフリー版の方が便利なため）
```

### **推奨ロジックの設計**

```
推奨判定フロー：

車両にハンズフリーPBDが装備されているか？
    ↓
YES → ハンズフリーPBDを推奨（R039-R041）
    ↓
NO → 通常のPBDが装備されているか？
    ↓
    YES → 通常のPBDを推奨（R015, R021, R028）
    ↓
    NO → 推奨しない
```
