# セールス支援（商談最適化）アルゴリズム - 設計書

**作成日：** 2026年1月15日  
**バージョン：** 1.0

---

## 📋 概要

顧客プロファイル、家族構成、用途、予算、商談シーンに応じて、最適な商談アプローチと納車時の説明レベルを自動判定するアルゴリズムです。

---

## 🎯 目的

### **課題**
- 顧客によって必要な情報量や説明の詳しさが異なる
- 初心者には丁寧すぎる説明が必要だが、熟練者には煩わしい
- 商談シーンごとに重点ポイントが変わる
- 納車時の説明が画一的で、顧客の理解度に合っていない

### **解決策**
- 顧客特性に応じた最適な商談トークスクリプトを提供
- 説明の詳細度を自動調整
- 商談シーンごとの重点ポイントを明確化
- 納車時のチェックリストを顧客レベルに応じて最適化

---

## 📊 判定ロジック

### **総合提案レベルの計算**

```
総合提案レベル = 顧客プロファイル補正値 × 
                 家族構成補正値 × 
                 用途補正値 × 
                 予算補正値 × 
                 商談シーン補正値
```

### **提案レベルと対応**

| 総合提案レベル | 説明スタイル | 所要時間 | 資料の詳しさ | デモ・体験 |
|--------------|------------|---------|------------|-----------|
| 2.0以上 | 最詳細 | 長め | 詳細資料 | 必須・丁寧 |
| 1.5〜2.0未満 | 詳細 | やや長め | 標準+詳細 | 推奨 |
| 1.0〜1.5未満 | 標準 | 標準 | 標準資料 | 任意 |
| 1.0未満 | 簡潔 | 短め | 簡潔資料 | ポイントのみ |

## 📊 具体的な商談例

### **例1：初めての車購入・単身・通勤・エントリー価格・初回来店**

**入力：**
- 顧客プロファイル：PROF_001（初めての車購入）→ 1.5
- 家族構成：FAM_001（単身）→ 1.0
- 用途：USE_001（通勤メイン）→ 1.0
- 予算：BUD_001（150万円以下）→ 1.2
- 商談シーン：SCENE_001（初回来店）→ 1.4

**計算：**
```
総合提案レベル = 1.5 × 1.0 × 1.0 × 1.2 × 1.4 = 2.52
```

**提案内容：**
- **説明スタイル：** 最詳細（基礎から丁寧に）
- **推奨車種：** アクアG（200万円）
- **推奨理由：** 初めての車購入で予算150万円以下。燃費が良く維持費が安いエントリーモデルが最適
- **推奨オプション：** BSM（5万円）
- **オプション理由：** 初心者には死角検知が安全性向上に有効
- **トークスクリプト：** 「初めての車購入ですね。まずは車の基本的な使い方から、安心してお使いいただけるようご説明させていただきます。アクアGは燃費も良く、運転しやすいコンパクトカーです。」
- **重点訴求：** 燃費の良さ、維持費の安さ、運転のしやすさ、安全装備の充実
- **提示資料：** 初心者向けパンフレット、維持費シミュレーション、安全装備動画
- **デモ・体験：** 駐車支援システムのデモ、スマホ連携体験、BSMの実演
- **次ステップ：** 試乗の提案、購入後サポート説明
