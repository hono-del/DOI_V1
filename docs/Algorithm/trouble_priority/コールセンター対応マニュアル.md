# トラブル対応優先度判定 - コールセンター対応マニュアル

## 📋 概要

このマニュアルは、車両トラブル発生時にコールセンターオペレーターが適切な対応を行うための指針を提供します。各判定IDに基づき、状況サマリーと具体的な対応指示を明示しています。

---

## 🎯 コールセンターの役割

### **1. 緊急度判定と初動対応**
- システムが自動判定した優先度ランクを確認
- 顧客の安全を最優先に初動対応を実施
- 必要に応じてロードサービスや緊急サービスを手配

### **2. 顧客サポート**
- 顧客の不安を軽減し、冷静な対応を促す
- 状況を正確に把握し、適切な指示を提供
- 対応完了までフォローアップを継続

### **3. リソース調整**
- ロードサービス車両の最適配置
- ディーラーや修理工場との連携
- 代替交通手段の手配

---

## 🚨 優先度ランク別の対応方針

### **最高優先度（スコア1.5以上）**

#### **対応方針**
- **即座の対応開始**（30秒以内）
- **多重リソースの動員**（ロードサービス、警察、救急等）
- **継続的な電話サポート**（対応完了まで電話を切らない）

#### **標準対応フロー**
1. ✅ 顧客の安全確認（怪我の有無、安全な場所にいるか）
2. ✅ 緊急ロードサービスを即座に手配
3. ✅ 必要に応じて110番または119番に通報
4. ✅ 代替交通手段の手配開始
5. ✅ 関係者（会社、家族）への連絡
6. ✅ 到着予定時刻を1分ごとに更新

#### **注意事項**
- 顧客を一人にしない（電話を切らない）
- パニック状態の顧客には落ち着いて指示
- 二次災害（後続車両衝突等）の防止を最優先

---

### **高優先度（スコア0.7〜1.49）**

#### **対応方針**
- **迅速な対応開始**（2分以内）
- **ロードサービスの手配**（20〜60分以内）
- **定期的なフォローアップ**（5〜10分ごと）

#### **標準対応フロー**
1. ✅ 顧客の状況確認（走行中か停車中か、同乗者の有無）
2. ✅ ロードサービスを手配（優先度に応じて20〜60分以内）
3. ✅ 安全確保の指示（停車位置、ハザードランプ等）
4. ✅ 到着予定時刻を伝達
5. ✅ 必要機材を確認してロードサービスに伝達
6. ✅ 到着までの待機方法を案内

#### **注意事項**
- 高齢者や初心者ドライバーには丁寧な説明
- 子供や妊婦同乗の場合は特別配慮
- 走行中のトラブルは安全停車を最優先指示

---

### **中優先度（スコア0.3〜0.69）**

#### **対応方針**
- **計画的な対応開始**（30分〜2時間以内）
- **症状確認とアドバイス**
- **ディーラー予約またはロードサービス手配**

#### **標準対応フロー**
1. ✅ 症状チェックリスト結果を確認
2. ✅ トラブルの原因を推定
3. ✅ 対応方法を提案（ロードサービス、ディーラー予約等）
4. ✅ 応急処置方法を案内（可能な場合）
5. ✅ 代替手段を提案（他の機能、代替交通等）
6. ✅ フォローアップの時間を約束

#### **注意事項**
- 緊急性は低いが、放置すると悪化する可能性を説明
- 当日中の対応を推奨
- 特定の状況（雨天、夜間等）での注意事項を伝達

---

### **低優先度（スコア0.29以下）**

#### **対応方針**
- **計画的な修理案内**（翌日可）
- **ディーラー予約のサポート**
- **予防保全のアドバイス**

#### **標準対応フロー**
1. ✅ トラブルの内容を確認
2. ✅ 緊急性が低いことを説明
3. ✅ ディーラーまたは修理工場を案内
4. ✅ 予約をサポート（翌日または都合の良い日時）
5. ✅ 注意事項を伝達（避けるべき走行条件等）
6. ✅ 予約日のリマインダー設定

#### **注意事項**
- 緊急性は低いが、修理は必要であることを強調
- 特定の条件下で危険になる可能性を説明
- 予防保全の重要性を伝達

## 📱 対応チャネル別の指示

### **緊急ロードサービス（最高・高優先度）**

#### **手配時の確認事項**
- 📍 **正確な位置情報**（GPS座標、住所、ランドマーク）
- 🚗 **車両情報**（車種、色、ナンバー）
- 👥 **顧客情報**（人数、年齢層、特別配慮の必要性）
- 🔧 **必要機材**（診断機器、交換部品、牽引装備等）
- ⏰ **推奨到着時間**（優先度に基づく）

#### **ロードサービスへの伝達事項**
```
【優先度：最高】
【トラブル：走行中エンジン停止】
【位置：○○高速道路上り線 ○○IC付近】
【顧客：妊婦運転、子供同乗】
【状況：路肩停車済み、ハザード点灯中】
【指示：最優先で派遣、到着予定時刻を1分ごとに更新】
```

---

### **通常ロードサービス（高優先度）**

#### **手配時の確認事項**
- 📍 位置情報
- 🚗 車両情報
- 🔧 必要機材
- ⏰ 推奨到着時間（20〜60分）

#### **ロードサービスへの伝達事項**
```
【優先度：高】
【トラブル：エンジン警告灯点灯】
【位置：○○市○○町○○駐車場】
【顧客：単独運転、日中】
【状況：駐車中、エンジン始動可能】
【指示：30分以内に診断機器持参で派遣】
```

---

### **コールセンター電話対応（中・低優先度）**

#### **電話対応時の確認事項**
- 📋 **症状の詳細**（いつから、どんな症状か）
- 🚗 **車両の状態**（走行可能か、異音・異臭の有無）
- 📅 **最終点検日**（直近の整備履歴）
- 🗓️ **顧客の都合**（修理可能な日時）

#### **電話対応の流れ**
1. **挨拶と状況確認**
   ```
   「お電話ありがとうございます。○○サポートセンターです。
   システムで△△のトラブルを検知しましたが、現在の状況を教えていただけますか？」
   ```

2. **症状の詳細確認**
   ```
   「症状はいつから始まりましたか？」
   「どのような音（臭い）がしますか？」
   「現在、車は走行可能ですか？」
   ```

3. **対応方法の提案**
   ```
   「承知いたしました。この症状は△△の可能性が高いです。
   当日中にディーラーでの点検をおすすめします。
   最寄りのディーラーをご案内いたしますので、予約をお取りしましょうか？」
   ```

4. **注意事項の伝達**
   ```
   「修理までの間、以下の点にご注意ください：
   ・雨天時の運転は避けてください
   ・夜間の走行は控えてください
   ・異音が大きくなった場合は、すぐにご連絡ください」
   ```

5. **フォローアップの約束**
   ```
   「修理完了後、○○時頃にお電話でフォローアップさせていただきます。
   何かご不明点はございますか？」
   ```

---

### **ディーラー予約サポート（低優先度）**

#### **予約時の確認事項**
- 🗓️ **顧客の希望日時**
- 📍 **最寄りのディーラー**（GPS自動検索）
- 🔧 **修理内容**（トラブルの種類）
- ⏰ **所要時間の目安**

#### **予約の流れ**
1. **最寄りディーラーの案内**
   ```
   「お客様の現在地から最寄りのディーラーは、
   ○○ディーラー（○○市○○町、お車で約15分）です。
   営業時間は9:00〜18:00です。」
   ```

2. **空き状況の確認**
   ```
   「本日は○○時と○○時に空きがございます。
   明日は終日空きがございます。
   いつがご都合よろしいでしょうか？」
   ```

3. **予約確定**
   ```
   「では、明日○月○日の○○時でご予約をお取りいたします。
   修理には約○○時間かかる見込みです。
   予約確認のメールをお送りいたします。」
   ```

4. **リマインダー設定**
   ```
   「予約日の前日にリマインダーをお送りいたします。
   何かご変更があれば、いつでもご連絡ください。」
   ```

## 🎨 顧客タイプ別の対応配慮

### **高齢者ドライバー**
- ✅ **ゆっくり、はっきり、丁寧に**説明
- ✅ 専門用語を避け、分かりやすい言葉を使用
- ✅ 指示は段階的に、一つずつ
- ✅ 不安を軽減する声かけ
- ✅ 必要に応じて家族への連絡を提案

### **初心者・ペーパードライバー**
- ✅ 安全確保の手順を詳細に説明
- ✅ パニックを防ぐための落ち着いた声かけ
- ✅ 簡単な手順から説明
- ✅ 電話を切らずにサポート継続
- ✅ 到着予定時刻を頻繁に伝達

### **妊婦ドライバー**
- ✅ 体調の優先確認
- ✅ 医療サポートの準備（必要に応じて）
- ✅ 代替交通手段の優先手配
- ✅ 家族への連絡を提案
- ✅ 安全で快適な待機場所の確保

### **子供同乗**
- ✅ 子供の安全を最優先
- ✅ 車内待機の安全確保（空調、照明等）
- ✅ 子供の不安軽減のためのアドバイス
- ✅ 到着時間を明確に伝達
- ✅ 必要に応じて親族への連絡

### **業務移動中**
- ✅ 会社への連絡を提案（必要に応じて代行）
- ✅ 代替交通手段の迅速な手配
- ✅ 業務への影響を最小化する提案
- ✅ 修理完了予定時刻の明確な伝達
- ✅ 社用車の場合は会社への報告サポート

## 📊 対応品質の管理

### **KPI（重要業績評価指標）**

| 指標 | 目標値 | 測定方法 |
|-----|-------|---------|
| **初回応答時間** | 30秒以内 | システム自動計測 |
| **対応開始時間** | 優先度に応じた時間内 | システム自動計測 |
| **顧客満足度** | 4.5/5.0以上 | 対応後アンケート |
| **解決率** | 95%以上 | 初回対応での解決割合 |
| **エスカレーション率** | 5%以下 | 上位者への引き継ぎ割合 |

### **通話品質チェックリスト**

#### **開始時**
- [ ] 明るく、親しみやすい挨拶
- [ ] システムが検知したトラブルの確認
- [ ] 顧客の安全状態の確認

#### **対応中**
- [ ] 顧客の話を最後まで聞く
- [ ] 共感の言葉を適切に使用
- [ ] 専門用語を避け、分かりやすく説明
- [ ] 対応方法を複数提案（可能な場合）
- [ ] 到着時間や対応時間を明確に伝達

#### **終了時**
- [ ] 対応内容のサマリー
- [ ] 次のステップの確認
- [ ] フォローアップの約束
- [ ] 他に困っていることがないか確認
- [ ] 感謝の言葉で締めくくり

## 🔄 特殊ケースの対応

### **トラブル頻発ユーザー**
- 📋 過去のトラブル履歴を確認
- 🔍 根本原因の特定を提案
- 🛠️ 包括的な点検を推奨
- 📞 定期的なフォローアップを提案

### **レンタカー・社用車**
- 🏢 レンタカー会社または会社への連絡
- 📋 契約内容の確認（保険、ロードサービス範囲）
- 💰 費用負担の確認
- 📝 報告書作成のサポート

### **深夜・早朝のトラブル**
- 🌙 深夜対応スタッフの優先派遣
- 🏨 必要に応じて宿泊施設を案内
- 👨‍👩‍👧 家族への連絡を提案
- 🚖 深夜タクシーの手配

### **山間部・遠隔地のトラブル**
- 📍 GPS位置情報を正確に把握
- ⏰ 到着時間が長くなることを説明
- 🗺️ ランドマークの確認
- 📞 電話サポートを継続
- 🚁 極端な場合はヘリコプター等も検討

---

## ✅ まとめ

コールセンターオペレーターの役割は：

1. ✅ **迅速かつ正確な状況把握**
2. ✅ **顧客の安全を最優先した対応**
3. ✅ **適切なリソースの手配**
4. ✅ **顧客の不安軽減と安心提供**
5. ✅ **対応完了までの継続的なサポート**

このマニュアルに従い、すべての顧客に最高品質のサポートを提供しましょう。
