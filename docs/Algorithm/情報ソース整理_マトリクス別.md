# 5つのマトリクス - 情報ソース整理

**作成日：** 2026年1月15日  
**バージョン：** 1.0

---

## 📋 概要

5つのマトリクスから出力される情報について、その引き出し先（取扱説明書 / 別ソース / 不明）を整理します。

---

## 🔍 マトリクス別の情報ソース整理

### **Excel 1：機能推奨マトリクス**

#### **アウトプット情報**
- 推奨する機能
- 機能の説明
- 使い方の手順
- 注意事項

#### **情報ソース分類**

| 情報項目 | 引き出し先 | 詳細 | 備考 |
|---------|----------|------|------|
| **機能名称** | 取扱説明書 | 各機能の正式名称 | P.200-250（安全装備）、P.150-180（ナビ・オーディオ） |
| **機能の説明** | 取扱説明書 | 機能の概要、目的 | 各機能のページに記載 |
| **操作手順** | 取扱説明書 | ボタン位置、操作方法 | 図解付きで記載 |
| **作動条件** | 取扱説明書 | 速度、天候等の条件 | 注意事項として記載 |
| **注意事項** | 取扱説明書 | 使用上の注意 | 警告・注意マークで記載 |
| **推奨シーン** | **別ソース** | 高速道路、雨天、夜間等 | **マーケティング資料、営業資料** |
| **メリット** | **別ソース** | 安全性向上、疲労軽減等 | **商品カタログ、プロモーション資料** |
| **顧客への説明トーク** | **引き出し先不明** | わかりやすい説明文 | **新規作成が必要** |

## 🔧 新規作成が必要な情報の作成方法

### **1. レベル別説明文（Excel 3）**

#### **作成方法**
```
取扱説明書の標準説明文を元に、6レベルに変換

最詳細（初心者）：
- 専門用語を避ける
- ステップを細かく分解
- 図解を多用
- 「なぜそうするか」を説明

↓

最簡潔（上級者）：
- 専門用語OK
- 要点のみ
- 箇条書き
- 結論から
```

#### **具体例：ACCの説明**

**取扱説明書（標準）：**
```
ACC（アダプティブクルーズコントロール）は、
先行車との車間距離を保ちながら自動で速度を調整します。
```

**最詳細（レベル1）：**
```
ACC（アダプティブクルーズコントロール）とは、
前の車との距離を自動で保ってくれる便利な機能です。

高速道路で使うと、アクセルを踏み続けなくても
自動で速度を調整してくれるので、
長距離運転の疲れが大幅に軽減されます。

使い方：
1. ステアリングの左側にあるACCボタンを押します
2. 希望の速度を設定します
3. 前の車との距離を3段階から選びます
4. あとは自動で速度を調整してくれます

注意：手放し運転はできません。
必ずステアリングを持って、前方を注視してください。
```

**最簡潔（レベル6）：**
```
ACC：先行車追従機能。
設定速度・車間距離で自動走行。
60km/h以上で作動。
```

### **2. 警告メッセージ（3ステップ）（Excel 3）**

#### **作成方法**
```
取扱説明書のトラブル情報を元に、
3ステップ構造に変換

ステップ1：状態理解
- 何が起きているか
- 現在の状況

ステップ2：判断支援
- 危険度の説明
- 走行可否

ステップ3：行動誘導
- 具体的な行動指示
- 連絡先案内
```

#### **具体例：ブレーキ警告灯**

**取扱説明書（標準）：**
```
ブレーキ警告灯が点灯した場合は、
ブレーキ系統に異常がある可能性があります。
直ちに安全な場所に停車し、
販売店にご連絡ください。
```

**3ステップメッセージ：**

**ステップ1：状態理解**
```
⚠️ ブレーキ警告灯が点灯しています

ブレーキ系統に異常が発生している可能性があります。
ブレーキの効きが悪くなっているかもしれません。
```

**ステップ2：判断支援**
```
🚨 危険度：最高

このまま走行を続けると、
ブレーキが効かなくなり、
重大な事故につながる恐れがあります。
```

**ステップ3：行動誘導**
```
📍 今すぐ行うこと

1. 速度を落とし、安全な場所に停車してください
2. エンジンを停止してください
3. 以下に連絡してください

📞 緊急ロードサービス：0120-XXX-XXX
📞 販売店：0120-YYY-YYY

⚠️ 走行は絶対にしないでください
```

### **3. 予測アクション・メッセージ（Excel 5）**

#### **作成方法**
```
対応履歴データベースから学習

1. 過去の対応パターンを分析
2. 顧客ステージ別の傾向を抽出
3. 次に発生しやすいアクションを予測
4. テンプレートメッセージを生成
```

#### **具体例：納車後1週間**

**対応履歴から学習：**
```
納車後1週間の顧客の70%が、
基本操作について問い合わせしている

主な問い合わせ内容：
- エアコンの使い方（30%）
- ナビの目的地設定（25%）
- スマホ接続方法（20%）
- 給油口の開け方（15%）
- その他（10%）
```

**予測アクション：**
