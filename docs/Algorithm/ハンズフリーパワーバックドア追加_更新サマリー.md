## 🔄 推奨ロジックへの影響

### **判定フロー**

```
ハンズフリーパワーバックドアの推奨判定
    ↓
車両個体ID取得（例：VEH_011）
    ↓
車種グレードID取得（CAR_004）
    ↓
車種機能マスタで装備状況確認
    ↓
    △（オプション）
    ↓
メーカーオプションマスタで装着確認
    ↓
    ├─ VEH_011（装着あり）→ ✅ 推奨
    └─ VEH_012（装着なし）→ ❌ 推奨しない
```

## 💡 ハンズフリーパワーバックドアの特徴

### **通常のパワーバックドアとの違い**

| 項目 | パワーバックドア（FN009） | ハンズフリーパワーバックドア（FN016） |
|-----|----------------------|--------------------------------|
| **操作方法** | スマートキー長押しまたは車内ボタン | リアバンパー下に足をかざす |
| **両手がふさがっている時** | 操作が困難 | 操作可能 |
| **利便性** | 高い | 非常に高い |
| **装備状況** | 多くの車種に標準装備 | 上位グレードまたはオプション |

### **推奨シーン**

✅ **買い物時**
- 荷物で両手がふさがっている時に便利

✅ **子育て中**
- 子供を抱っこしている時でも操作可能

✅ **雨天時**
- 傘をさしている時でも操作可能
