## 🚀 HTMLへの変換方法

### **方法1：npxを使う（推奨・インストール不要）**

```powershell
# Presentationフォルダに移動
cd docs/Presentation

# npxでHTMLに変換
npx @marp-team/marp-cli マトリクス説明_プレゼンテーション.md -o マトリクス説明_プレゼンテーション.html --html --allow-local-files
```

### **方法2：Marp CLIをインストールして使う**

```powershell
# Marp CLIをグローバルインストール
npm install -g @marp-team/marp-cli

# HTMLに変換
marp マトリクス説明_プレゼンテーション.md -o マトリクス説明_プレゼンテーション.html --html --allow-local-files
```

### **方法3：PowerShellスクリプトを使う**

```powershell
# Presentationフォルダに移動
cd docs/Presentation

# スクリプトを実行
.\convert_to_html.ps1
```

## 📝 編集方法

### **Markdownファイルの編集**

`マトリクス説明_プレゼンテーション.md` を直接編集してください。

#### **スライドの区切り**
```markdown
---
```

#### **2カラムレイアウト**
```markdown
<div class="columns">
<div>
左側の内容
</div>
<div>
右側の内容
</div>
</div>
```

#### **ボックス（強調表示）**
```markdown
<div class="box">
強調したい内容
</div>
```

#### **カラーテキスト**
```markdown
<span class="success">+40%</span>  <!-- 緑色 -->
<span class="warning">-20%</span>  <!-- 赤色 -->
```

## 🔧 トラブルシューティング

### **npxコマンドが見つからない**

Node.jsをインストールしてください：
https://nodejs.org/

### **変換エラーが出る**

1. ファイル名に日本語が含まれている場合、パスの指定に注意
2. `--allow-local-files` オプションを追加
3. Markdownの構文エラーをチェック

### **スタイルが反映されない**

Markdown内の `style:` セクションを確認してください。
