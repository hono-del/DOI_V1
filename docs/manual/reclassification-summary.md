## 2. 改善内容

### 2.1 実施した改善

1. **キーワードリストの拡充**
   - 各カテゴリのキーワードを拡充
   - より多くのテキストを適切に分類

2. **セクションID単位での一貫性調整**
   - セクション内の50%以上が同じカテゴリの場合、すべてをそのカテゴリに統一
   - 562件を一貫性調整

3. **要素タイプによる補助分類**
   - `attention_para`要素をカテゴリ②に優先分類
   - `steptxt`, `callouttxt`, `itemtxt`要素をカテゴリ①に優先分類

### 2.2 改善効果

- **デフォルト分類の削減**: 42.7% → 18.9%（23.8ポイント改善）
- **セクション単位の一貫性**: 562件を調整
- **分類の均等化**: 各カテゴリの分布がより均等に

### 6.3 分類精度90%達成のための必要情報

**最優先（即座に実施）**:
1. **誤分類サンプルの抽出**（最低100件）
   - カテゴリ⑦（デフォルト分類）に分類されたテキストの分析
   - 各カテゴリで「不適切」と判断されるテキストの抽出
2. **キーワードリストの拡充**
   - 誤分類サンプルからキーワード候補を抽出
   - 各カテゴリ50-100語の追加

**高優先（1週間以内）**:
3. **セクションIDパターンの拡充**
   - 全セクションIDのリストとカテゴリ分布の分析
   - セクションIDとカテゴリの対応表の作成
4. **要素タイプパターンの拡充**
   - 全要素タイプのリストとカテゴリ分布の分析
   - 要素タイプとカテゴリの対応表の作成

**期待される効果**:
- キーワードリストの拡充: 分類精度 +3-4%
- セクションID・要素タイプパターンの拡充: 分類精度 +1-2%
- コンテキスト情報の活用: 分類精度 +1-2%
